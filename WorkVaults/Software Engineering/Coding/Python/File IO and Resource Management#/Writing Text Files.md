open() Modes



| Mode    | Meaning |
| ----------- | ----------- |
| 'r'      | open for reading |
| 'w'   | open for writing |
| 'a' | open for appending |
| Selector | Meaning |
| 'b' | binary mode |
| 't' | text mode |


wite() returns the number of codepoints written. Don't sum these values to determine file length.
